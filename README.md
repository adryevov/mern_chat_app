How to start app
Front-end
1. Open new terminal
2. In root folder run 'npm install' 
3. Run 'npm start' in terminal
4. Open 'http://localhost:3000'

Back-end
1. Open separate terminal 
2. 'cd backend' 
3. run 'npm install'
4. run 'node server.js' ( will run on localhost:4000)
